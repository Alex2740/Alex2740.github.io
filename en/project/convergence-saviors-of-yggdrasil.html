Convergence: Saviors of Yggrasil

Project Status On hold Project Type University / group project Project Duration ~3 months Software Used Unity, Aseprite Languages Used C# Primary Role(s) Lead programmer / Artist

About Convergence: Saviors of Yggrasil

Convergence: Saviors of Yggdrasil is a project I did during my classes at UQAC. There were 4 people in the project. The game was created until a playable prototype was produced. This project helped me to increase my knowledge of Unity and C#.

Tilemap system

With another person of the group, we implemented the tilemaps system. This system allows to contain all the information of the terrain (obstacles, units, ...). The system uses an overlay of tilemaps to create a hierarchy and separate the elements of the map (ground, walls, obstacles, decoration, ...). The system uses Unity Tilemaps to create the board object that the state machine will use to move and control the units.

Input system

I implemented an input system that controlled the state machine. The system uses events to communicate with the state machine. Using events allows to support keyboard or mouse controls. The game can be played with keyboard only or mouse only.

Camera system

I implemented the camera system of the game. I started to add a functionality to move the camera. The camera can be moved by the player using the arrow keys. Then, I added a zoom feature, now the player can have an overview of the map.

After implementing these features, I realized that the player could lose his character on the map. So I added a tracking system: the camera follows the currently selected character.

After I added a system to change colors when the camera goes in the other world. A blue filter is added when the camera is in Midgard.

Teleportation mechanic

I implemented the teleportation mechanic. A character can be teleported if the tile in the other world is not already occupied. Also, we decided to allow the teleportation to Midgard immediatly to the player. Teleporting in the other way has a channeling. It is more difficult to return to the world of the gods. During the channeling, a blue totem appears to prevent a unit from blocking the arrival tile.

Art

The group didn't have an artist to create the design, so I created all the graphic assets of the game. For the style, I started with pixelart because it was the style I mastered the most. I was inspired by the graphic direction of the game Forager. I used the Aseprite software to create the spritessheets and tilesheets.
